-- SUMMARY --

Chosen uses the Chosen jQuery plugin to make your <select> elements more user-friendly.


-- INSTALLATION --

  1. Download the Chosen jQuery plugin (http://harvesthq.github.com/chosen/) and extract the file under sites/all/libraries.
  2. Download and install jQuery Update (6.x-3.x) which will update your Drupal to jQuery 1.7.
       The 6.x-3.x branch is currently only available in a sandbox:
       - http://drupal.org/sandbox/aidan/1367896
       You can download the latest 6.x-3.x-dev from:
       - https://github.com/aidanlister/jquery_update/zipball/6.x-3.x
  3. Download and enable the module.
  4. Configure Chosen at Administer > Site Configuration > Chosen (requires administer site configuration permission)
